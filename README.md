# Anchor Skeleton

This is the official skeleton for the **Anchor Framework**.

## Quick Start (Managed)

```bash
composer create-project beniyke/anchor-skeleton my-app
cd my-app
php dock
```

## Quick Start (Standalone)

Download this repository and run:
```bash
php dock
```
